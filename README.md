# monitorprofiler
rivy/monitorprofiler fork

Controls monitors
- brightness / contrast / RGB gain
- sharpness / volume
- change source
- put in standby / power off
- reset factory defaults

- "hidden" hotkey to power on all monitors : winkey + 0 (won't work on windows login screen)
